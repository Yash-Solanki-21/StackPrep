# Job Preparation Platform

A full-stack Job Preparation Platform built using React, Node.js, Express, and MongoDB.

## Features

- User Authentication (JWT)
- Resume Analyzer (ATS Score)
- Coding Practice
- MCQ Test
- Admin Panel
- Protected Routes

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- JWT
- Monaco Editor

## Installation

### Frontend

npm install
npm run dev

### Backend

npm install
npm run dev

## Environment Variables

Create a .env file in the server folder.

PORT=
MONGO_URI=
JWT_SECRET=
